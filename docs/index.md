# Android Knowledge Hub

Tài liệu tổng hợp kiến thức Android theo từng nhóm chủ đề để dễ học, dễ tra cứu và dễ cập nhật.

## Nội dung chính

- Nền tảng Kotlin và tư duy lập trình
- Android Core
- UI: XML và Jetpack Compose
- Architecture
- Data layer
- Performance
- Testing
- Interview notes

## Cách dùng

Chọn chủ đề ở sidebar để đọc nhanh theo từng nhóm.

## Nguyên tắc cập nhật

- Mỗi bài chỉ tập trung vào 1 chủ đề
- Có phần: khái niệm, ví dụ, best practices, lỗi thường gặp
- Ưu tiên ngắn gọn, dễ tra cứu
